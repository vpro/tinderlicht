# Tinderlicht

First:
``` npm install ```

Copy `config.example.json` to `config.json` and adjust all the values.

Backend (the secure connection to Firebase):
``` npm run backend ```

Deploy:
Creates a deployable bundle.
``` npm run deploy ```


## debugging purposes

Webpack test server:
``` npm run dev ```


Output debug messages of the backend server
``` npm run backend:dev ```
